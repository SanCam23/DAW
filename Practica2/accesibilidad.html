Compromiso con la Accesibilidad

Nuestro objetivo es que cualquier usuario pueda navegar, leer y utilizar nuestra web de forma cómoda. Por ello, hemos implementado estilos alternativos que mejoran la experiencia de uso para distintos niveles de visión.

Estilos Accesibles Disponibles

1. Modo Oscuro

Este estilo invierte los colores del fondo y el texto, reduciendo la fatiga visual y mejorando la lectura en entornos con poca luz. Fondo oscuro y texto claro proporcionan un alto nivel de comodidad visual.

2. Alto Contraste

El estilo de alto contraste mejora la legibilidad al aumentar la diferencia entre el color del texto y el fondo. Este modo es útil para usuarios con discapacidad visual o daltonismo.

3. Letra Grande

El estilo de letra grande incrementa el tamaño del texto y el espaciado entre líneas para facilitar la lectura a personas con baja visión.

4. Alto Contraste con Letra Grande

Este modo combina ambos ajustes para ofrecer una experiencia óptima a usuarios que necesitan simultáneamente un texto más grande y mayor contraste de color.

Cómo Activar un Estilo Accesible

En navegadores como Mozilla Firefox, los usuarios pueden seleccionar un estilo alternativo desde el menú Ver → Estilo de página. También se puede modificar automáticamente mediante un selector de estilo dentro de la propia web.
